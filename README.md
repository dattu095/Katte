# Katte

## Dependencies
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

## Installation
1. UV Installation
``` bash
uv run main.py
```

2. PIP Installation
``` bash
python -m venv .venv        # Create venv
source .venv/bin/activate   # Activate Virtual-Environment
pip install -e .            # Install Dependencies

python main.py
```
